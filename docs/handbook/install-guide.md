# SpeakLow 安装指南

按住热键说话，松开后文字自动输入到光标位置。

## 系统要求

- macOS 13.0 (Ventura) 或更高版本
- 支持 Intel 和 Apple Silicon

## 安装步骤

### 1. 打开 DMG

双击 `SpeakLow.dmg`，会弹出一个窗口，里面有 SpeakLow 图标和 Applications 文件夹快捷方式。

### 2. 拖入 Applications

把 **SpeakLow.app** 拖到 **Applications** 文件夹上松开。

### 3. 首次启动

由于这个 app 没有 Apple 开发者签名，macOS 会阻止运行。按以下步骤解决：

**方法一（推荐）：**
1. 打开 **访达 → 应用程序**，找到 SpeakLow
2. **右键点击**（或按住 Control 点击）SpeakLow.app
3. 选择 **「打开」**
4. 弹窗提示「无法验证开发者」，点击 **「打开」**

**方法二：**
1. 双击 SpeakLow，看到 "无法打开" 的提示后点「好」
2. 打开 **系统设置 → 隐私与安全性**
3. 往下滚动，会看到「已阻止 SpeakLow」的提示，点击 **「仍要打开」**
4. 输入密码确认

> 只有首次启动需要这样做，之后直接双击就能打开。

### 4. 授权系统权限

SpeakLow 需要两个系统权限才能正常工作：

#### 麦克风权限
首次录音时，系统会弹窗请求麦克风权限，点击 **「好」** 即可。

#### 辅助功能权限（重要！）
这个权限用于在光标位置插入识别出的文字。**如果不授权，文字只能复制到剪贴板，无法自动插入。**

1. 打开 **系统设置 → 隐私与安全性 → 辅助功能**
2. 点击左下角的 🔒 锁图标解锁
3. 点击 **「+」**，找到并添加 **SpeakLow.app**
4. 确保 SpeakLow 旁边的开关是 **打开** 状态

> 如果你之前装过旧版本，建议先删掉旧的权限条目，再重新添加。

### 5. 配置 API Key

SpeakLow 通过阿里云**百炼平台**的语音识别服务进行转写，需要填入 API Key 才能正常使用。

**首次启动时**，如果检测到未配置 API Key，app 会自动弹出密钥设置页面。

手动配置步骤：

1. 前往 [百炼平台](https://bailian.console.aliyun.com/) 获取 API Key（注意：不是阿里云主账号 AK，也不是 Coding Plan）
2. 点击菜单栏图标 → **设置 → 密钥** tab
3. 在输入框中粘贴 API Key，点击 **保存**
4. 保存后**立即生效**，无需重启 app

> 如果按热键后提示"请先配置 API Key"，说明当前未正确配置，请检查密钥设置。

## 使用方法

1. SpeakLow 启动后会出现在 **菜单栏**（屏幕右上角小图标）
2. **按住右 Option 键**（⌥，键盘右侧）开始录音，屏幕顶部会出现波形动画
3. **松开**后自动识别并输入文字到当前光标位置
4. 点击菜单栏图标可以修改设置（热键、麦克风、LLM 润色等）

### 热键选项

在菜单栏设置中可以选择：
- **右 Option 键**（默认）
- **Fn 键**
- **F5 键**

## 常见问题

### "SpeakLow 已损坏，无法打开"

这是 macOS Gatekeeper 的限制。打开终端，执行：

```bash
xattr -cr /Applications/SpeakLow.app
```

然后重新打开 SpeakLow。

### 文字没有插入到光标位置

1. 检查辅助功能权限是否已授权（见上方步骤）
2. 如果已授权仍不行，去 **系统设置 → 隐私与安全性 → 辅助功能**，关掉 SpeakLow 的开关再打开
3. 部分应用（如 VS Code、Slack 等 Electron 应用）会走剪贴板粘贴方式，这是正常的

### 录音没有反应

1. 检查麦克风权限是否已授权
2. 检查菜单栏设置中选择的麦克风是否正确
3. 菜单栏图标点击后如果显示 "Bridge 未运行"，尝试退出并重新打开 SpeakLow

### 菜单栏看不到图标

SpeakLow 的图标可能被其他图标挤到了「更多」区域。按住 ⌘ 拖动菜单栏图标可以调整位置。

## 卸载

1. 退出 SpeakLow（菜单栏图标 → 退出）
2. 从「应用程序」文件夹删除 SpeakLow.app
3. 可选：删除 **系统设置 → 隐私与安全性 → 辅助功能** 中的 SpeakLow 条目
