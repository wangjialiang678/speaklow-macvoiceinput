# SpeakLow App Icon 设计文档

## 品牌分析

### 产品定位
- **名称**: SpeakLow（轻声说）
- **功能**: macOS 语音转文字输入工具，按住热键录音，松开后将识别文字插入光标位置
- **用户**: 开发者、写作者、效率工具用户
- **调性**: 低调、高效、专业、安静（名字暗示"轻声"）

### 名称含义
"Speak Low" 源自对低调、安静的暗示。不是大声呼喊的语音助手，而是安静高效的输入工具——轻声说，文字到。

### 核心隐喻候选
1. **声波 + 文字** — 语音转文字的核心动作
2. **麦克风** — 录音的直觉符号
3. **低语/耳语** — 呼应 "Speak Low" 的名字
4. **波纹** — 声音在空气中的传播，安静优雅
5. **羽毛/叶子 + 声波** — 轻盈 + 声音

## 设计规范

### Apple 平台要求
- **macOS**: 1024x1024 PNG，系统自动裁切圆角矩形
- **图标风格**: macOS 偏质感/光影层次（与 iOS 扁平不同）
- **缩放测试**: 16px favicon 仍需可辨识
- **安全区**: 边缘留 ~10% 避免裁切

### 设计原则
- 单一焦点：一个核心图形元素
- 避免文字：缩小后不可读
- 高对比度：深色/浅色 Dock 背景都好看
- 独特性：避免与系统 app 或常见语音 app 撞脸

## 10 个方案

### 方案 1: 声波圆环
深蓝渐变背景，中心一个抽象声波图形形成圆环，线条细腻优雅。macOS 风格带微妙光影。

### 方案 2: 低语麦克风
暗色背景上一个极简麦克风剪影，麦克风右侧散发柔和的声波涟漪，整体安静优雅。

### 方案 3: 羽毛声波
一片羽毛或叶子造型，纹理由声波线条构成。暗示"轻"和"声音"的双重含义。薄荷绿/青色系。

### 方案 4: 字母 S 声波
字母 "S"（SpeakLow 首字母）由声波曲线构成，形成优美的字母标识。紫色渐变。

### 方案 5: 涟漪水滴
一滴水落入平静水面产生同心涟漪——暗示"轻声"的触发效果。蓝色/青色。

### 方案 6: 对话气泡 + 波形
圆角对话气泡内包含一段音频波形图。气泡造型简洁，波形表达语音。暖橙色。

### 方案 7: 月亮耳语
新月造型与声波结合——月亮暗示安静/夜晚/低声。深蓝+金色。

### 方案 8: 极简波形
纯粹的 3-4 条竖线组成音频波形，居中放置。极简主义。黑白/灰调。

### 方案 9: 折纸鸟 + 声波
一只折纸风格的小鸟正在轻声歌唱，嘴边有小声波。几何感+有温度。青色/珊瑚色。

### 方案 10: 光谱圆盘
圆形内排列渐变色的频谱柱状图，形成 360 度环形频谱。科技感强。彩虹渐变。

## 生成参数

所有方案统一使用：
- 尺寸：1024x1024
- 比例：1:1
- 背景：纯色或渐变（不透明，Apple 要求）
- 风格关键词：`macOS app icon, rounded square, professional, clean, 3D render, subtle lighting`

## 评选标准

1. 16px 缩放辨识度
2. Dock 中与其他 icon 的协调度
3. 品牌故事传达力（能否让人联想到"语音输入"）
4. 独特性（不与常见语音 app 撞脸）
5. 美观度
