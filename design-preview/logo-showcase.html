Usage Guidelines

Spacing & Clear Space

  • Minimum clear space: Equal to the height of the microphone stand on all sides
  • Minimum size (mark only): 20px — below this, use the favicon variant
  • Minimum size (combo mark): 120px wide to maintain legibility of letterforms
  • Preferred background: Void black (#0D0F12) or Warm white (#E8E6E3)
  • Don't rotate: The mark is optimized for upright orientation only
  • Don't recolor: Use the approved palette; no arbitrary tints or gradients
  • Don't stretch: Always scale proportionally using the viewBox
  • macOS menu bar: Use favicon at 18–22px with template image mode (single color)
